### ■ 주요 특징
* #### 온라인 쇼핑몰의 전반적인 기능을 모두 구현.
* #### 관리자의 관점으로 쇼핑몰 운영에 필요한 여러가지 기능을 구현.
* #### 상품 검색 · 상세검색과 많은 페이지에서의 다양한 조건 검색과 그에 따른 페이징을 구현.
* #### 게시판의 페이징을 MyBatis와 JPA 두 종류로 구현.
* #### 다양한 Open API를 이용하여 여러가지 기능 구현.
